This directory holds custom notification sounds for the Android app.

## urgent_alarm.mp3
Place a loud, distinct alarm MP3 file here named `urgent_alarm.mp3` to use as the 
urgent/emergency lead notification sound.

Requirements:
- Format: MP3 or OGG
- Duration: 3-8 seconds recommended
- Name must be `urgent_alarm` (extension stripped by Android)
- Should be louder / more attention-grabbing than the default notification sound

If no file is placed here, the system alarm ringtone is used as a fallback 
(which is already louder than the default notification sound).
